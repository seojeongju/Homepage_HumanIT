🎯 도메인 연결 간단 가이드 (humanit.ai.kr)
═══════════════════════════════════════════════════════════════

📍 현재 상황:
   • 도메인: humanit.ai.kr (가비아에서 구매)
   • 호스팅: Cloudflare Pages
   • 프로젝트: humanit-webapp

───────────────────────────────────────────────────────────────

STEP 1️⃣: Cloudflare 설정 (5분)

1. https://dash.cloudflare.com/ 접속

2. Workers & Pages → humanit-webapp 선택

3. Custom domains 탭 → Set up a custom domain

4. humanit.ai.kr 입력 → Continue

5. 완료! (다음 단계로)

───────────────────────────────────────────────────────────────

STEP 2️⃣: 가비아 설정 (10분)

방법 선택:
  A. 네임서버 변경 (더 쉬움! 권장)
  B. DNS 레코드 추가 (네임서버 유지)

─────────────────────
방법 A: 네임서버 변경
─────────────────────

1. Cloudflare 메인 → Add a site → humanit.ai.kr

2. Free 플랜 선택

3. 네임서버 주소 메모:
   예) alex.ns.cloudflare.com
      roxy.ns.cloudflare.com

4. https://www.gabia.com/ 로그인

5. My가비아 → 서비스 관리 → 도메인 → humanit.ai.kr 관리

6. 네임서버 설정 → 네임서버 설정 변경

7. Cloudflare 네임서버 입력 → 확인

─────────────────────
방법 B: DNS 레코드 추가
─────────────────────

1. https://www.gabia.com/ 로그인

2. My가비아 → 서비스 관리 → 도메인 → humanit.ai.kr 관리

3. DNS 설정 → 레코드 추가

4. 다음 정보 입력:
   ┌─────────────────────────────────┐
   │ 타입: CNAME                     │
   │ 호스트: @                       │
   │ 값: humanit-webapp.pages.dev    │
   │ TTL: 3600                       │
   └─────────────────────────────────┘

5. 저장/적용

───────────────────────────────────────────────────────────────

STEP 3️⃣: 대기 (1-2시간)

⏰ DNS 전파 대기:
   • 최소: 10분
   • 보통: 1-2시간
   • 최대: 48시간

🔍 확인 방법:
   https://dnschecker.org/
   → humanit.ai.kr 입력
   → CNAME 레코드 확인

───────────────────────────────────────────────────────────────

STEP 4️⃣: 완료 확인

1. https://humanit.ai.kr 접속

2. 🔒 자물쇠 아이콘 확인 (HTTPS)

3. 모바일에서도 테스트

4. 완료! 🎉

───────────────────────────────────────────────────────────────

❓ 문제 발생 시

문제: 사이트를 찾을 수 없음
해결: DNS 설정 재확인, 1시간 더 대기

문제: 리디렉션 루프
해결: Cloudflare SSL 설정 → "Full"로 변경

문제: 안전하지 않음 (HTTPS 안됨)
해결: DNS 전파 완료 대기
